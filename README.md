• **📌 FPGA Pong 프로젝트**
1972년 아타리 Pong의 하드웨어 설계 철학을 FPGA 상에서 재현한 디지털 시스템으로,  
TTL 기반 아날로그 회로의 동작 원리를 Verilog로 복원하고 현대적인 디지털 구조를 적용하여 물리 엔진과 게임 로직을 확장한 프로젝트

---

**⚙️ 세부 기능**

- *모드 선택 시스템*
  - 스위치 입력에 따라 `1:1 모드 / AI 모드 / 예측 모델 모드` 전환

- *AI 상대 모드*
  - 학습된 데이터를 기반으로 패들의 위치를 제어하여 자동 대응

- *예측 모델 모드*
  - 공의 현재 위치 및 이동 방향을 계산하여  
    패들이 충돌 예상 지점으로 이동하도록 제어

- *게임 메커니즘*
  - 게임 시작 시 공이 랜덤 방향으로 생성 및 이동
  - 패들 충돌 시 반대 방향으로 반사
  - 미충돌 시 상대방 점수 증가

- *점수 및 종료 조건*
  - 일정 점수 도달 시 게임 종료
  - 종료 후 초기 상태로 복귀 가능

---

**🔴 구현 한계 및 개선 사항**
- *미니 LCD 하드웨어 제약*
  - 비표준 해상도 지원 장비(ADC 케이블) 수급 문제로 디스플레이 확장 제한

- *AI 학습 구조 한계*
  - 게임 중 상태 정지(Freeze)가 불가능하여 학습 데이터 수집 및 정확도 개선에 어려움

- *예측 기반 AI 밸런스 문제*
  - 공 궤적을 완벽히 예측하여 게임 난이도가 비정상적으로 상승  
  → 플레이어 입력 이후 공이 반응하도록 설계하여 실수 유도 방식으로 개선
